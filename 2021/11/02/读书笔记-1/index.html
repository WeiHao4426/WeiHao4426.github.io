人类一败涂地

发表于 2021-11-02 更新于 2023-06-11 分类于 游戏策划 阅读次数：

正好最近万圣节促销，买了人类一败涂地这款游戏，正在熟悉操作并过关，那就结合我游玩的过程说一下它的关卡设计吧。

首先，这款游戏的操作教学都是融入到前几个关卡的。新玩家在初次进入游戏时，直接控制一个小白人，然后依靠自己的经验去进行摁下开关、搬动箱子到开关上、拉下扳手等等操作。这些都是游戏的基本操作，玩家是完全可以根据自己的日常生活经验来推断出来的。

然而，完成这些基础的教学关卡后，玩家会遭遇第一个难题：火车关（至少对于我是难题）。玩家需要爬上一个火车，然后一步步通过火车登上平台。对我来说，这一关有点复杂了。我甚至不清楚“将视角向下拉可以爬上平台”这样的基础操作（前面的关卡没教），这个关卡确实是一个难度的提升。当我查询百度上的教程，并最终通过捷径（翻墙）到达下一关时，我才获得了相关操作的知识。

第一关：豪宅

第一关豪宅是新手引导关，关卡难度并不难。
在第一关中，玩家主要需要了解移动视角、按下按钮、以及使用黄色长方体来获取教学的各种简单操作。
关卡流程：

  1. 进入到豪宅关卡的场景，地面上有一个遥控器。拿起遥控器后，场景内会出现一个幕布，上面会出现教程动画
  2. 顺着路来到一扇门前，这是一扇常见的按键门，将视野对准门边的按键按下抓取键即可开门。
  3. 进入按键门顺着小道走，捡起这里的遥控器，出现教程动画：如果想要拿取较高处或者低处的物品，需要调整视野中心位置的高低。
  4. 来到如上开关门前，此处有上下两个开关，调整视角分别按下两个开关即可开门。
  5. 继续前进来到关卡出口处，此时按键门的按键有两个，水平分布。站在开关前同时按下左右手抓取按键即可同时按下开关，出口处的门会被打开，走近门从出口跃下即可结束本关卡

第二关：火车以及第三关：搬运第四关：山峰

对我这个新手来说，这几关的难度明显提升很多，我花了好多时间才通过。
在这几关中，玩家需要自己领悟：

  • 移开箱子发现门
  • 爬到火车上并前进到平台上（如果没有找到门）
  • 站在箱子上，抓住柱子并用脚移动车厢
  • 以及其他基本的结合性操作
  • 举起双手冲刺以爬上更高的平台
  • 甚至实在过不去时，在网上搜索翻墙的方法（我属于这种）

这一关除了让玩家把之前的基础操作结合起来，更重要的是让玩家熟悉这个游戏的谜题设计套路。这个套路是贯穿这个游戏所有关卡的。它包括：

把门隐藏到某个东西后面
推某个有轮子的东西，产生变化
某个开关需要重物在上面压着
按下按钮
站在跷跷板上，用自己的质量使跷跷板运动起来
（大概这么多了）

所以，第一关是让玩家熟悉基本的操作，那么第二关就是让玩家体会游戏中谜题的设计，熟悉游戏谜题设计者的思路，达到上手游戏的效果。

第五关：拆除

在前几关，玩家基本懂了游戏的基本操作和迷题的设计思路。此时，玩家将面对几个连续的解密小关卡。它们组合在一起，就形成了第一个正式的关卡。

这个关卡包括以下部分：

  1. 第一个场景内有一个被木条封住的门，抓住木条向后拽即可拆除木条，拆掉后进入门。
  2. 第二个场景，房间内有一扇落地窗，先将落地窗前的摇杆拉至最右侧，窗外吊着的重物会向右侧移动利用自重砸开墙壁。双手将墙角的灭火器举高高冲向落地窗，即可将落地窗砸碎来到屋外。
  3. 通过碎裂的墙壁进入到下一个场景，跃起抱住铁球利用惯性适时松开抓取键即可荡到对面。
  4. 【然后我就卡在这里了......等我打过这一关再说吧。

以上，游戏”人类一败涂地”的新手关卡及第四关梳理。
